# Java Animal Counter GUI
**Course:** CIS 407 - Java Programming II
**Grade:** A

##  The Challenge
The objective was to create a robust, user-friendly interface to manage and count different animal types, requiring complex event handling and data validation.

## Technical Implementation
**Language:** Java
**Framework:** Swing (GUI)
**Build Tool:** Maven
**Key Features:** Custom exception handling for data entry and polymorphic class structures.

## Key Lesson
During this project, I encountered challenges with Maven dependency conflicts. Resolving these taught me the importance of build automation and environment consistency-skills I now apply when thinking about cloud infrastructure.
