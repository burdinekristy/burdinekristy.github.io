# Electronic Arts: Software Engineering Simulation
**Provider:** Forage | **Completed:** September 2025

## Simulation Overview
This program involved completing professional-level tasks modeled after the daily responsibilities of a Software Engineer at Electronic Arts.

## Tasks Completed

**Game Object Class:** Designed and Implemented a robust structure for game entities.
**Inventory System Improvement:** Refactored code to enhance performance and scalability.
**Live Bugfix:** Identified and resolved critical logic errors in a simulated production environment.

## Professional Growth
This experience bridged the gap between my academic Java studies and real-world engineering. It taught me how to navigate existing codebases and provide value through both new features and maintenance.
